# 🎬 Movie Browsing App

## 📌 Description

This is a Movie Browsing web application where users can explore and discover movies easily. The app allows users to search movies by name, filter them based on genres, and sort them according to rating or popularity. It also includes pagination to navigate through multiple pages of movie results efficiently.

## 🚀 Features

* Search movies by name
* Filter movies by genre
* Sort movies based on rating and popularity
* Pagination for browsing multiple pages
* Responsive and user-friendly UI

## 🌐 API Used

* JSON Fakery Movies API
* https://jsonfakery.com/movies/paginated

(Note: This API provides mock movie data for frontend development purposes.)

## 🛠️ Tech Stack

* React (Vite)
* JavaScript
* CSS / Tailwind

## ⚙️ Setup Instructions

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
